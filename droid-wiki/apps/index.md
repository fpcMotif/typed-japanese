# Apps

Active contributors: Yifeng Wang

Typed Japanese ships one deployable app: the interactive playground. Everything else in the repository is a library, documentation, or learning material.

| Page | What it covers |
| --- | --- |
| [Playground](playground.md) | The Vite + React app that hosts the grammar course, analyzer, and glossary |

The playground is deployed to GitHub Pages at `https://typedgrammar.github.io/typed-japanese/` on every push to `main`. See the [deployment](../deployment.md) page for the CI configuration.
