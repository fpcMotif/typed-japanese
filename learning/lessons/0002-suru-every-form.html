🌸 Typed Japanese · Lesson 2

する through every form

The most-used verb in Japanese, seen whole. It's irregular — so the type system stores it in a hand-written table, which you'll now read directly.

Big idea this time: the library gives you a base, and you add a small tail (ます・ない・る・ば) to finish the everyday word. し ＋ ます ＝ します.

Remember from Lesson 1: する is one of only two irregular verbs. So ConjugateVerb<する, F> doesn't compute a rule — it just looks the answer up in a table someone wrote by hand. Here is that exact table.¹

1 Everyday forms

The five you'll hit first. The middle column is literally what the type returns; the next is what you actually say.

FormType returnsYou sayMeaning
辞書形dictionary"する"するsurudoes / to do
ます形polite"し"しますshimasudoes (polite)
て形te-form"して"してshitedoing; and do; (＋ください) please do
た形past"した"したshitadid
ない形negative"し"しないshinaidoes not
Notice ます形 and ない形 both return the same base し — the difference is only the tail you attach (ます vs ない). Meanwhile て形 and た形 come back already finished.

2 Wider forms

The other seven. These unlock "can / is done / make do / let's / do it! / if".

FormType returnsYou sayMeaning
可能形potential"でき"できるdekirucan do
受身形passive"され"されるsareruis done
使役形causative"させ"させるsaserumake / let do
意向形volitional"しよう"しようshiyōlet's do / shall do
命令形imperative"しろ"しろshirodo it! (formal: せよ)
条件形 / 仮定形conditional"すれ"すればsurebaif (one) does
可能形 is special: する's "can do" is the separate word できる (not しれる). The library knows this — it returns でき. Also 条件形 and 仮定形 share one base, すれ; add ば for the everyday "if".

3 Base ＋ tail = the word

This is the pattern that ties the whole table together. The type gives the left side; you add the right.

し ＋ ます  →  します   (polite)
し ＋ ない  →  しない   (negative)
でき ＋ る  →  できる   (potential)
させ ＋ る  →  させる   (causative)
すれ ＋ ば  →  すれば   (conditional)

And in the type system, that "add a tail" step is just another template literal — exactly the `${Base}${Tail}` tool from Lesson 1:

type する = IrregularVerb & { dictionary: "する" };

type Base   = ConjugateVerb<する, "ます形">;  // "し"
type Polite = `${Base}ます`;                // "します"  ✓ you just conjugated by hand

🎉 Why this verb is worth the effort

する is the busiest verb in Japanese: stick it onto a noun and you get a new verb, for free.

勉強 (study) ＋ する → 勉強する benkyō suru, "to study".   料理 (cooking) ＋ する → 料理する ryōri suru, "to cook".²
Learn する's table once, and all of those conjugate the same way.
