Glossary

The shared vocabulary for these lessons. Both worlds — TypeScript and Japanese — meet here. Skim it; return to it. This is the canon: lessons will use these exact terms.

TS  TypeScript — the type machinery

TypeA description of the *shape* of a value, checked by the compiler before the program runs. In this project, types are the whole show — there is no runtime code.
Literal typeA type that is one exact value. "話す" is a type that only the string "話す" satisfies. This is how Japanese words become types.
Union type"One of these." "う" | "く" | "す" — used to list the legal godan endings.
Generic / type parameterA type that takes inputs, like a function takes arguments. ConjugateVerb<V, F> — V and F are the inputs.
Constraint (extends, in <>)Limits what an input may be. F extends ConjugationForm means "F must be one of the legal forms."
Conditional type (extends ? :)Type-level if / else. T extends U ? X : Y = "if T fits U, then X, else Y." This is how the rules branch.
Template literal typeString-building in types: `${A}${B}` glues type-strings together. This is how a stem + an ending becomes a finished word.
Mapped type / lookupAn object-shaped type used as a table. Map[ending][form] looks up the right conjugation piece.
inferPattern-matching inside a conditional type — pulls a piece *out* of a string-type (e.g. the part of "walk" before an ending). Covered in a later lesson.
Type-level functionInformal name for a generic that maps inputs to an output type. Conjugation is exactly this.

日  Japanese — the grammar

Stem (語幹)The unchanging front part of a verb. In 食べる the stem is 食べ.
EndingThe final kana that changes during conjugation. In 話す the ending is す.
Godan verb (五段動詞)"Five-row" verb (a.k.a. Group 1 / u-verb). Ends in う,く,ぐ,す,つ,ぬ,ぶ,む,る. The ending shifts across the あいうえお vowel rows. Most verbs are these.
Ichidan verb (一段動詞)"One-row" verb (a.k.a. Group 2 / ru-verb). Always ends in る; you just drop る and add the form. Simplest to conjugate.
Irregular verb (不規則)Only two: する (to do) and 来る／くる (to come). Memorised, not derived.
Conjugation form (活用形)One way a verb can be inflected — tense, politeness, mood. This project supports 12 (see below).
ParticleA small grammatical marker attached to words (よ, か, の, なら …). Adds role or nuance.

形  The conjugation forms (活用形)

辞書形Dictionary (plain) form — the base, e.g. 話す.
ます形Polite form — 話し(ます).
て形Te-form — connective "and / then", 話して.
た形Past form — 話した.
ない形Negative form — 話さ(ない).
可能形Potential — "can do", 話せ(る).
受身形Passive — "is done".
使役形Causative — "make/let do".
意向形Volitional — "let's / shall", 話そ(う).
命令形Imperative — command, 話せ.
条件形 / 仮定形Conditional / hypothetical — "if".
